### 内容简介
本书深入探讨了行为金融学在量化投资中的应用，结合了心理学、经济学和市场微观结构理论，为量化策略开发提供了新的视角。

### 核心章节
1. 行为金融学基础
2. 市场异象分析
3. 投资者心理偏差
4. 行为因子构建
5. 策略实现与优化
